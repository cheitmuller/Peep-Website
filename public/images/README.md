# Brand Assets

This directory contains brand assets for the Peep website.

## Required Assets

### Logo Files
- **peep-logo.svg** - Main Peep bird logo (SVG format, preferred)
- **peep-logo.png** - PNG fallback of the logo (if needed)
- **apple-icon.svg** - Apple App Store icon for download buttons

### Specifications
- **Logo**: Black line weight, white or transparent background, ~32-40px height for header
- **Format**: SVG preferred for scalability
- **Colors**: Should match brand guidelines (black lines on white/transparent)
